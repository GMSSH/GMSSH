# GMSSH私有部署说明

GMSSH私有部署提供在线版和离线版本，在线版访问官网：
https://www.gmssh.com/private

离线版本说明：
## 环境要求
- **操作系统**: Linux Server
- **内核版本**: 3.10+

## 安装步骤

1. **下载**: 在外网环境下载最新的 stable 版本安装包。
2. **传输**: 将安装包传输到离线部署服务器。
3. **解压**: 在安装目录下执行以下命令解压压缩包：
   ```bash
   tar xzf <安装包>.tar.gz -C .
   ```

## 配置与管理

### 端口配置
服务默认使用 **TCP 80 端口**。

若端口冲突，可执行以下命令更新服务端口，并自动启动服务：
```bash
bash ./deploy.sh setport <端口号>
```
> [!IMPORTANT]
> 注意：在设置端口时一定要跟端口号！

### 服务管理

- **启动服务**:
  若服务未启动，在安装包目录下执行：
  ```bash
  bash ./deploy.sh start
  ```

- **查看状态**:
  执行以下命令查看服务状态：
  ```bash
  bash ./deploy.sh status
  ```

- **初始化服务**:
  启动服务时可能需要先初始化服务，执行：
  ```bash
  bash ./deploy.sh init
  ```
  初始化完成后再启动服务。

> [!NOTE]
> 以上命令执行可能需要 **root 权限** 或 **sudo 权限**。

## 网络安全

请确保防火墙或安全组已放开相应的 TCP 端口规则（默认为 80 端口，或您修改后的端口）。

## 访问服务

当服务状态正常时，通过浏览器访问：
`http://服务器 IP: <端口号>`
